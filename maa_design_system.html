Rules
— No gradients. No glassmorphism. No blur. Flat solid fills only.
— Every surface uses exactly one of: White, Ivory, Blush, Sage, or Charcoal.
— All borders are 1.5px solid #2C2C2C — no lighter, no thinner.
— All shadows are 2px 2px 0px #2C2C2C (offset only, no blur) — ink-print feel.
— Active states: replace White background with Blush. Never change border color.
— Error states: use Charcoal text + dashed border. No red.

Rules
— Headings, affirmations, and stat numbers always use Playfair Display (serif).
— All UI text, body copy, buttons, labels use DM Sans (sans-serif).
— Never use font-weight above 600. Body is 400, UI labels are 500, display is 600.
— Section labels are UPPERCASE + 1px letter-spacing in DM Sans 500 13px.
— Minimum font size: 11px (captions). Never below.
